## Requirements

* Node.js 22
* TypeScript 5.5
* yarnpkg v4
* Docker 4.32


## How to run

- Install dependencies
```bash
$ yarn install
```
- Do copy `example.env` to `.env`, fill it with actual values
- Run Docker Compose
```bash
$ docker compose up
```
- Run demo
```bash
$ yarn start
```
